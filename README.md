# 🏦 XBank

Aplicación bancaria desarrollada con React y Firebase que permite registrar usuarios, iniciar sesión, consultar el saldo, realizar transferencias entre cuentas y visualizar el historial de movimientos en tiempo real.

---

# 📌 Características

- ✅ Registro de usuarios
- ✅ Inicio de sesión
- ✅ Cierre de sesión
- ✅ Saldo inicial de $100.000
- ✅ Transferencias entre usuarios
- ✅ Actualización automática del saldo
- ✅ Historial de movimientos
- ✅ Interfaz moderna y responsive
- ✅ Base de datos en Firebase Firestore

---

# 🚀 Tecnologías utilizadas

- React
- Vite
- Firebase Authentication
- Firebase Firestore
- React Icons
- CSS
